Section 3.2 Quantifiers Part II

In this section we will look at how to negate statements involving quantifiers.

If you look back at the Check your Understanding questions in the previous section, you should notice that both \(\forall x\in D, P(x)\) and \(\forall x\in D, \sim P(x)\) were false, which means they are not negations of each other. Similarly, both \(\exists x\in D, P(x)\) and \(\exists x \in D, \sim P(x)\) were true. In general, the two statements won't necessarily have the same truth values, but the examples were chosen to make sure we can see that they are not negations of each other.

Negating Quantified Statements:

The negation of \(\forall x\in D, P(x)\) is \(\exists x\in D, \sim P(x)\text{;}\)

or, in notation, \(\sim(\forall x\in D, P(x))\equiv \exists x\in D, \sim P(x)\text{.}\)

The negation of \(\exists x\in D, P(x)\) is \(\forall x\in D, \sim P(x)\text{;}\)

or, in notation, \(\sim(\exists x\in D, P(x))\equiv \forall x\in D, \sim P(x)\text{.}\)

We can think of negation as switching the quantifier and negating \(P(x)\text{,}\) but it will be really helpful if we can understand WHY this is the negation. Thinking about negating a “for all” statement, we need the statement to not be true for all things, which means it must be false for something, Thus, there exists something making \(\sim P(x)\) true. Thinking about negating a “there exists” statement, we need there not to exist anything making \(P(x)\) true, which means \(P(x)\) must be false for everything. Thus, everything makes \(\sim P(x)\) true.

Example 3.2.1. Negating a Quantified Statement.

Negate the statement: For all primes \(p\text{,}\) \(p\) is odd. Is this statement true or false?

Answer 1

There exist a prime \(p\) such that \(p\) is not odd. The original statement is false, since we can find an even prime (\(p=2\)).

Negate the statement: There exists a real number \(x\text{,}\) such that \(x^2< 0\text{.}\) Is this statement true or false?

Answer 2

For all real numbers \(x\text{,}\) \(x^2\geq 0\text{.}\) The original statement is false, since the negation is true.

Many of our quantified statements may have predicates involving other logical connectives. So it is going to be important to remember how to negate "and"s, "or"s, and "if...then"s.

Reminder: negating logical statements.

  • AND statements.
    \(\sim(p\wedge q)\equiv \sim p\ \vee \sim q\text{.}\) This is DeMorgan's Law.
  • OR statements.
    \(\sim(p\vee q)\equiv \sim p\ \wedge \sim q\text{.}\) This is DeMorgan's Law.
  • IF...THEN statements.
    \(\sim(p\rightarrow q)\equiv p\ \wedge \sim q\text{.}\) The negation of a conditional is NOT a conditional.

Negating a universal conditional statement:

\begin{equation*} \sim(\forall x\in D, P(x)\rightarrow Q(x))\equiv \exists x\in D, \sim(P(x)\rightarrow Q(x))\\ \equiv \exists x\in D, P(x)\wedge \sim Q(x) \end{equation*}

Contrapositive of a universal conditional statement:

\(\forall x\in D, P(x)\rightarrow Q(x)\) has contrapositive \(\forall x\in D, \sim Q(x)\rightarrow \sim P(x)\text{.}\)

Similarly the converse of \(\forall x\in D, P(x)\rightarrow Q(x)\) is \(\forall x\in D, Q(x)\rightarrow P(x)\text{.}\)

The relationship between “for all” and “there exists” can be used to show some surprising things.

Vacuously true statements:

What happens if our domain, \(D\text{,}\) has nothing in it? In particular, let \(D=\emptyset\text{,}\) the empty set. Is \(\forall x\in D, P(x)\) true or false? Well, let's look at the negation: \(\exists x\in D, \sim P(x)\text{.}\) Now the negation MUST be false since \(D\) has nothing in it, so there can't exist something in \(D\) making \(\sim P(x)\) true. Since the negation is false, the original statement is true! We say \(\forall x\in D, P(x)\) is vacuously true.

Example 3.2.2. Vacuously True Statement.

Consider the statement “For all llamas, \(L\text{,}\) in Discrete Math, \(L\) is getting an A.” The negation is “There exists a llama, \(L\text{,}\) in Discrete Math, such that \(L\) is not getting an A.” Since no such llama exists, the negation is false. Making the original true. So every llama in Discrete is getting an A.

As one additional note, it can be helpful in deciding if your negation is correct by finding the truth value of both the original and the negation. They should have opposite values. Similarly, if you need to determine the truth value of a complex statement, it might be easier to find the truth value of the negation.

Reading Questions Check Your Understanding

1.

Write the negation of \(\forall n\in \mathbb{Z}, n>0\text{.}\)

Answer
\(\exists n\in \mathbb{Z}, n\leq 0.\)
2.

Write the negation of \(\forall n\in \mathbb{Z}\text{,}\) if \(n>0\) then \(n^3>0\text{.}\)

Answer
\(\exists n\in \mathbb{Z}\text{,}\) \(n>0\) and \(n^3\leq 0.\)
3.

Write the negation of \(\exists n\in \mathbb{Z}\text{,}\) \(n>5\) and \(n^2\leq 10\text{.}\)

Answer
\(\forall n\in \mathbb{Z}\text{,}\) \(n\leq 5\) or \(n^2> 10.\)
4.

Write the negation of \(\exists n\in \mathbb{Z}\text{,}\) such that if \(n^2>4\) then \(n>2\) or \(n<-2.\)

Answer
\(\forall n\in \mathbb{Z}\text{,}\) \(n^2>4\) and \(n\leq 2\) and \(n\geq -2.\) Note, we can also write this as \(\forall n\in \mathbb{Z}\text{,}\) \(n^2>4\) and \(-2\leq n\leq 2.\)
5.

Write the contrapositive of \(\forall n\in \mathbb{Z}\text{,}\) if \(n>0\) then \(n^3>0\text{.}\)

Answer
\(\forall n\in \mathbb{Z}\text{,}\) if \(n^3\leq 0\) then \(n\leq 0.\)
