Item 4.
If you need counterexamples that involve set complements, if might be helpful to define a small universal set such as \(U=\{1, 2, 3, 4, 5, 6\}\text{.}\) Then if \(A=\{1, 2, 3\}\text{,}\) \(A^C=\{4, 5, 6\}\text{,}\) which is easier to work with than, say, all the integers except 1, 2, 3.
