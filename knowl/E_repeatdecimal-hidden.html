Show the repeating decimal \(0.\overline{123}\) is a rational number.

We need to find \(a, b\in \mathbb{Z}\) such that \(0.\overline{123}=\frac{a}{b}\) and \(b\neq 0\text{.}\) This one is trickier than the last example and requires a new technique. First, let \(x=0.\overline{123}=0.123123\ldots\text{.}\) Then multiply both sides of \(x=0.123123\ldots\) by 1000, so that \(1000x=123.123123\ldots\text{.}\) We chose 1000 in order to get 1 set of the repeated digits in front of the decimal point. Now we subtract the two equations from each other:

\begin{equation*} 123.123123\ldots=1000x\\ -(0.123123\ldots=x)\\ \end{equation*}

Resulting in \(123=999x\text{.}\) Now we just solve for \(x\text{:}\) \(x=123/999\text{.}\)

Letting \(a=123, b=999\text{,}\) we can see that \(a, b\in \mathbb{Z}, b\neq 0\text{.}\)
