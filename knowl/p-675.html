The way to think about the Principle of Mathematical Induction is that if you know the statement is true for some starting value, (\(P(a)\) is true), and if you can show that knowing the statement is true for some value allows you to know it is true for the next value (\(P(k)\rightarrow P(k+1)\text{,}\) then you know it for all values greater than or equal to \(a\text{.}\)
