Notation: We will usually use capital letters for sets, such as \(S\) or \(A\text{.}\) If we want to talk about elements in a set \(S\text{,}\) we use the notation \(x\in S\text{.}\) We read this notatation as "\(x\) is in \(S\)" or "\(x\) is an element of \(S\)". If \(x\) is not in \(S\text{,}\) then we use the notation \(x\notin S\text{.}\)
