Let \(A=\mathbb{R}, B=\mathbb{R}\text{.}\) Let \(f(x)=x+2\text{.}\) Then \(f:\mathbb{R}\rightarrow\mathbb{R}\text{.}\) We can check that it is a function: if we choose an \(x\text{,}\) we can see that there is a corresponding real number \(x+2\text{,}\) which always exists. We can also see that for any \(x\text{,}\) there is only one possible number as output.
