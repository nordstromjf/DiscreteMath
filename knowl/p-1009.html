Advice for finding counterexamples:

  1. Write the negation of the statement so you can make sure your sets satify the appropriate conditions.
  2. Most counterexamples for the types of statements we see in this chapter can be found without needing to find "tricky" sets. So just try something, it will probably work.
  3. You might need to be careful about whether your sets intersect or not.
  4. If you need counterexamples that involve set complements, if might be helpful to define a small universal set such as \(U=\{1, 2, 3, 4, 5, 6\}\text{.}\) Then if \(A=\{1, 2, 3\}\text{,}\) \(A^C=\{4, 5, 6\}\text{,}\) which is easier to work with than, say, all the integers except 1, 2, 3.
