We can find all the integers that are even and \(4\leq n\leq 16\text{.}\) This is the set \(\{2, 4, 6, 8, 10, 12, 14, 16\}\text{.}\) For each of these numbers we can demonstrate a way to write them as the sum of primes: \(4=2+2, 6=3+3, 8=3+5, 10=3+7, 12=5+7, 14=7+7, 16=11+5\text{.}\)
